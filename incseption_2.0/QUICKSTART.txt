🎯 WHAT IS THIS?
================
A complete full-stack web application for tracking agricultural 
products from farm to consumer using Ethereum blockchain.

Problem: Agricultural supply chains lack transparency
Solution: Immutable blockchain records + QR code verification
Impact: Farmers get fair prices, consumers verify authenticity

✨ KEY FEATURES
===============
✅ Blockchain-powered supply chain tracking
✅ 4 user roles: Farmer, Distributor, Retailer, Consumer
✅ MetaMask wallet integration
✅ QR code product verification
✅ Real-time price tracking
✅ Complete transaction history
✅ Modern React UI with responsive design
✅ JWT authentication with role-based access
✅ MongoDB data persistence

🎓 DEMO WORKFLOW
================

1. Register 4 test accounts:
   - Account 1: Register as FARMER
   - Account 2: Register as DISTRIBUTOR
   - Account 3: Register as RETAILER
   - Account 4: Register as CONSUMER

2. Create a product (Farmer):
   - Click "Create Product"
   - Fill in: Name, Description, Quantity, Location
   - Approve MetaMask transaction

3. Transfer through supply chain:
   - Farmer: Transfer to Distributor (set price: 0.5 ETH)
   - Distributor: Transfer to Retailer (set price: 0.8 ETH)
   - Retailer: Sell to Consumer (set price: 1.2 ETH)

4. Verify product:
   - Consumer: View product, see complete history
   - Scan QR code to verify authenticity

💡 TROUBLESHOOTING
==================

Issue: MetaMask shows "No provider"
→ Make sure MetaMask is installed and contract address is in .env

Issue: "User already registered"
→ Each wallet address can only register once. Use a different account.

Issue: Frontend won't compile
→ Already fixed! Use: npm run dev:frontend

Issue: Port 3000 already in use
→ Use different port in vite.config.js
